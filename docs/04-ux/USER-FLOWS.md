## Flow 1: First Launch (Первый запуск)

**Триггер:** Пользователь скачал приложение из App Store и открыл впервые.
**Цель:** За <60 секунд показать ценность, собрать персонализацию, показать paywall.
**Связанные фичи:** F-005, F-008, F-007, F-010

### Happy Path

```
[App Store → Install → Open]
    ↓
Splash Screen (1 сек, логотип)
    ↓
Welcome Screen
  "Определи опасность за 3 секунды"
  [Demo: фото укуса → результат green]
  Прогресс-бар: 15%
  [Далее] / Skip
    ↓
Quiz Step 1: "Что вас беспокоит?"
  Мульти-выбор: укусы, пауки, змеи, клещи
  Прогресс-бар: 35%
  [Далее] / Skip
    ↓
Quiz Step 2: "Расскажите о себе"
  ZIP код (для гео-опасностей)
  Есть ли дети? (да/нет)
  Есть ли питомцы? (тип)
  Прогресс-бар: 55%
  [Далее] / Skip
    ↓
Social Proof Screen
  Рейтинг 4.7★ | "247,391 опасностей определено"
  2 реальных отзыва из App Store
  Прогресс-бар: 75%
  [Далее]
    ↓
Paywall Screen
  "Ваш персональный план безопасности готов"
  Анимированный paywall
  Годовой план первый: $29.99/год (badge "Лучшее предложение")
  Месячный план: $4.99/мес (за "Все планы")
  7-дневный бесплатный trial
  Таймлайн (триал → напоминание → списание)
  Free vs Premium таблица
  Прогресс-бар: 100%
  [Начать trial] / [Продолжить бесплатно]
    ↓
Home Screen (guest mode, 3 бесплатных скана)
    ↓
[Результат: пользователь на Home, готов к первому реальному скану]
```

### Error States

| Ситуация | Поведение |
|----------|----------|
| Нет интернета при первом запуске | Показать Welcome + demo результат (offline). Квиз и paywall — отложить до появления сети. Баннер "Без интернета — ограниченный режим" |
| Adapty не загрузился (paywall не готов) | Пропустить paywall → Home. Показать paywall позже при первой попытке premium-фичи |
| App Store прервал покупку (sandbox error) | Показать "Не удалось оформить подписку. Попробуйте позже" → [Повторить] / [Продолжить бесплатно] |
| Пользователь нажал "Restore Purchase" | Проверить Adapty → если подписка есть: разблокировать Premium. Если нет: "Подписка не найдена" |

### Edge Cases

| Ситуация | Поведение |
|----------|----------|
| Пользователь нажимает Skip на каждом шаге | Минимальный онбординг: Welcome → Paywall → Home. Персонализация = default (все категории, без гео) |
| Пользователь закрыл приложение на середине онбординга | При следующем запуске — продолжить с последнего шага (persist progress локально) |
| Пользователь уже был подписчиком (переустановка) | Restore Purchase на paywall → определить подписку → Home (Premium) |
| ZIP код не введён | Гео-опасности отключены, показывать общий контент. Запросить позже в Profile |

## Flow 2: Core Action — Danger Scan (Сканирование опасности)

**Триггер:** Пользователь нажал кнопку "Сканировать" на Home или в Tab Bar.
**Цель:** Фото → контекстные вопросы → результат с action card за ≤15 секунд.
**Связанные фичи:** F-001, F-002, F-003, F-004, F-010

### Happy Path

```
[Home Screen → Tap "Scan" button]
    ↓
Camera Permission Prompt (если первый раз)
  "Danger Scanner нужен доступ к камере для сканирования опасностей"
  [Разрешить] / [Не сейчас]
    ↓
Camera Screen
  Видоискатель (полный экран)
  Кнопка съёмки (центр)
  Кнопка галереи (выбрать из Camera Roll)
  Кнопка фонарика
  Подсказка: "Сфотографируйте объект крупным планом"
    ↓
[Tap: сделать фото]
    ↓
Photo Preview
  Превью сделанного фото
  [Использовать фото] / [Переснять]
    ↓
[Tap: Использовать фото]
    ↓
Analyzing Screen (2–4 сек)
  Анимация сканирования
  Social proof: "247K+ опасностей определено"
  Прогресс анимации
    ↓
Post-Scan Questions (2–3 вопроса, quick-tap)
  Вопросы адаптированы к категории:
  
  Укус: "Где на теле?" → "Как давно?" → "Есть отёк/покраснение?"
  Паук: "Размер?" → "Регион?" → "Где нашли?"
  Змея: "Регион?" → "Среда?" → "Был ли укус?"
  Клещ: "Где на теле?" → "Сколько времени?" → "Есть ли bullseye?"
  
  Каждый вопрос = 1 тап (варианты)
  [Пропустить] доступен
    ↓
Result Screen
  ┌─ Danger Level: RED / YELLOW / GREEN ─┐
  │ "Похоже на: [название]"               │
  │ Confidence: 87%                       │
  ├─ Action Card ─────────────────────────┤
  │ 1. Немедленные действия (first aid)   │
  │ 2. Когда к врачу (ER / urgent / нет)  │
  │ 3. Что НЕ делать                      │
  ├───────────────────────────────────────┤
  │ ⚠ Disclaimer: "Informational only,   │
  │ not medical advice"                   │
  ├───────────────────────────────────────┤
  │ [Связаться со специалистом]           │
  │ [Сохранить] [Поделиться]              │
  └───────────────────────────────────────┘
    ↓
[Результат: пользователь видит оценку опасности + план действий]
```

### Error States

| Ситуация | Поведение |
|----------|----------|
| Камера не разрешена | Показать экран "Для сканирования нужна камера" → [Открыть настройки] / [Выбрать из галереи] |
| Нет интернета | Показать "Для анализа нужен интернет" → [Повторить] / [Сохранить фото для анализа позже]. Фото сохраняется локально, скан запустится при появлении сети |
| AI timeout (>10 сек) | Показать "Анализ занимает больше времени..." → после 15 сек: "Не удалось проанализировать. Попробуйте переснять с лучшим освещением" → [Переснять] / [Повторить] |
| AI не может определить объект (confidence <50%) | Показать "Не удалось определить. Попробуйте:" + советы по фото (ближе, свет, фокус) → [Переснять]. НЕ показывать "green" — conservative bias (F-004) |
| Фото слишком тёмное/размытое | До отправки: "Фото слишком тёмное. Включите вспышку" → [Переснять с фонариком] |

### Edge Cases

| Ситуация | Поведение |
|----------|----------|
| Лимит 3 скана/мес исчерпан (Free) | Вместо Camera Screen → Metered Paywall: "Сканы на этот месяц закончились" → [Подписаться] / [Ждать до [дата]] |
| Первый скан вообще (guest, без регистрации) | Скан работает полностью. После результата → Push Permission + Registration Prompt: "Создайте аккаунт, чтобы сохранить результат" |
| Змея на фото (любая) | Conservative bias: ВСЕГДА "Если вас укусили — звоните 911". Danger level ≥ YELLOW. Никогда "неядовитая" (F-004) |
| Confidence <85% | Danger level ≥ YELLOW. Показать "Мы не уверены на 100% — будьте осторожны" (F-004) |
| Пользователь пропустил все вопросы | Результат показывается на основе только фото. Менее точный, но всегда ≥ YELLOW при низком confidence |
| Фото содержит лицо/не-опасность | "Мы анализируем опасности: укусы, пауков, змей, клещей. На фото не обнаружено объектов для анализа" → [Переснять] |

## Flow 3: Return Visit (Повторный визит)

**Триггер:** Пользователь открывает приложение повторно (D1+).
**Цель:** Быстрый доступ к скану + engagement через историю/алерты.
**Связанные фичи:** F-006, F-009, F-001

### Happy Path — Есть предыдущие сканы

```
[Open app]
    ↓
Home Screen
  ┌─ Верх ─────────────────────────────┐
  │ "Привет, Sarah"                     │
  │ Бесплатных сканов: 2/3 (Free)      │
  │ или "Premium ✓" (подписчик)         │
  ├─ Алерты (если есть) ───────────────┤
  │ 🔔 "Сезон клещей начинается       │
  │     в вашем регионе"                │
  ├─ Последние сканы ──────────────────┤
  │ [Фото] Укус паука | Yellow | 2 дня │
  │ [Фото] Клещ       | Red    | 5 дн  │
  ├─ Quick Actions ────────────────────┤
  │ [🔍 Сканировать] (CTA, крупная)   │
  └────────────────────────────────────┘
    ↓
[Tap: Сканировать] → Camera (Flow 2)
  или
[Tap: Последний скан] → Result Screen (сохранённый)
  или
[Tap: Алерт] → Информация о сезонной опасности
```

### Happy Path — Повторный визит из Push-уведомления

```
[Push: "Сезон клещей начинается в NC — защитите семью"]
    ↓
[Tap push]
    ↓
Seasonal Alert Screen
  Информация о сезоне клещей
  Региональная статистика
  Советы по профилактике
  [Сканировать клеща] → Camera (Flow 2)
```

### Error States

| Ситуация | Поведение |
|----------|----------|
| Токен авторизации истёк | Silent refresh через Supabase. Если неудача → Login Screen |
| Нет интернета | Показать Home с кешированными данными + баннер "Офлайн". Историю показывать из кеша. Кнопка "Сканировать" → предупреждение о необходимости интернета |
| Подписка истекла (был Premium → стал Free) | Баннер на Home: "Ваша подписка истекла" → [Продлить]. Лимит 3 скана/мес. Историю >5 показывать заблокированной |

### Edge Cases

| Ситуация | Поведение |
|----------|----------|
| Первый повторный визит, не было сканов | Home с empty state: "Сделайте первый скан!" + подсказки что можно сканировать (фото-примеры категорий) |
| Не завершённый скан (фото сохранено, но не отправлено) | Баннер: "У вас есть незавершённый скан" → [Продолжить анализ] |
| 30 дней с последнего скана плесени/трещины | Push reminder: "30 дней с последнего скана плесени — время проверить" → открывает History с последним скангом плесени |

## Flow 4: Upgrade Flow (Конверсия в Premium)

**Триггер:** Бесплатный пользователь достигает лимита или хочет premium-фичу.
**Цель:** Конвертировать в платного подписчика с минимальным фрикшном.
**Связанные фичи:** F-007

### Happy Path — Лимит сканов исчерпан

```
[Free user, 0 сканов осталось]
    ↓
[Tap: Сканировать]
    ↓
Metered Paywall
  "Сканы на апрель закончились (0/3)"
  "Разблокируйте неограниченные сканы"
  
  Что вы получите:
  ✓ Безлимитные сканы
  ✓ Полные Action Cards с first aid
  ✓ Вся история сканов
  ✓ Сезонные geo-alerts
  
  [Годовой: $29.99/год — "Лучшее предложение"]
  [Месячный: $4.99/мес]
  7-дневный бесплатный trial
  
  [Начать 7-дневный trial]
    ↓
Apple/Google Payment Sheet
    ↓
[Confirm payment]
    ↓
Success Screen
  "Добро пожаловать в Premium! 🛡"
  "Ваш trial до [дата]. Напомним за 1 день до списания."
  [Начать сканирование]
    ↓
Camera Screen → Flow 2
    ↓
[Результат: пользователь — Premium, неограниченные сканы]
```

### Happy Path — Premium-фича заблокирована

```
[Free user → History tab → видит заблокированные сканы (#6+)]
    ↓
[Tap на заблокированный скан]
    ↓
Contextual Paywall
  "Разблокируйте полную историю сканов"
  Список premium-преимуществ
  [Начать trial] / [Не сейчас]
    ↓
(аналогично Metered Paywall)
```

### Happy Path — Повторное предложение неконвертированному

```
[Free user, отказался от paywall ранее]
    ↓
[Через 3 дня: Alternative Offer]
  Push: "Специальное предложение: первый месяц за $0.99"
    ↓
[Tap push]
    ↓
Alternative Paywall
  Другой оффер: $0.99 первый месяц
  или: 14-дневный trial вместо 7
  [Попробовать] / [Нет, спасибо]
```

### Error States

| Ситуация | Поведение |
|----------|----------|
| Платёж отклонён (карта) | "Не удалось оформить подписку. Проверьте способ оплаты" → [Повторить] / [Продолжить бесплатно] |
| Apple/Google IAP недоступен | "Покупки временно недоступны. Попробуйте позже" → [OK] → Home (Free) |
| Пользователь отменил покупку на экране Apple Pay | Вернуть на paywall. Не повторять автоматически. Показать "Вы можете подписаться позже в Profile → Подписка" |
| Restore purchase не находит подписку | "Подписка не найдена. Убедитесь, что используете тот же Apple ID" → [Повторить] / [Связаться с поддержкой] |

### Edge Cases

| Ситуация | Поведение |
|----------|----------|
| Пользователь уже в trial, нажал "Subscribe" | Показать "У вас уже есть активный trial до [дата]" → [OK] |
| Family Sharing (iOS) | Если другой член семьи подписан → определить через Adapty → разблокировать Premium |
| Пользователь подписался через другое устройство | Restore Purchase при запуске → синхронизация через Adapty |
| Trial закончился, пользователь не заплатил | Push за 1 день до конца: "Ваш trial заканчивается завтра". При истечении → переход на Free, баннер "Продлите Premium" |

## Flow 5: Share Flow (Поделиться результатом)

**Триггер:** Пользователь хочет поделиться результатом скана (врачу, семье, соцсети).
**Цель:** Одним тапом создать shareable контент + вирусный потенциал.
**Связанные фичи:** F-003, F-006

### Happy Path

```
[Result Screen после скана]
    ↓
[Tap: "Поделиться"]
    ↓
Share Options
  ┌────────────────────────────────┐
  │ Поделиться результатом         │
  │                                │
  │ [📋 Для врача (PDF)]           │
  │ → Полный отчёт: фото + AI-    │
  │   оценка + danger level +     │
  │   рекомендации + disclaimer   │
  │                                │
  │ [📱 В соцсети / мессенджер]    │
  │ → Branded image: фото +       │
  │   danger level + "Checked     │
  │   with Danger Scanner"        │
  │                                │
  │ [📎 Скопировать результат]     │
  │ → Текст с описанием           │
  └────────────────────────────────┘
    ↓
[Tap: "В соцсети"]
    ↓
Branded Share Image (auto-generated)
  ┌────────────────────────────────┐
  │ [Фото объекта]                 │
  │                                │
  │ ⚠ YELLOW — Будьте осторожны   │
  │ "Похоже на: Brown Recluse"     │
  │                                │
  │ Checked with Danger Scanner 🛡 │
  │ dangerscanner.app              │
  └────────────────────────────────┘
    ↓
iOS Share Sheet (Messages, Instagram, WhatsApp...)
    ↓
[Результат: контент отправлен, получатель видит branded image + ссылка на приложение]
```

### Happy Path — Экспорт для врача

```
[Result Screen → "Поделиться" → "Для врача (PDF)"]
    ↓
PDF Generation (1–2 сек)
  Содержимое PDF:
  - Дата и время скана
  - Фото (полное разрешение)
  - AI-идентификация + confidence %
  - Danger level
  - Контекстные ответы пользователя
  - Рекомендации (action card)
  - Disclaimer
  - QR-код приложения
    ↓
iOS Share Sheet → Email / AirDrop / Files
    ↓
[Результат: врач получает структурированный отчёт]
```

### Error States

| Ситуация | Поведение |
|----------|----------|
| PDF-генерация не удалась | "Не удалось создать отчёт. Попробуйте поделиться изображением" → fallback на share image |
| Нет интернета | Share image генерируется локально → можно отправить через AirDrop / сохранить. PDF: "Нужен интернет для генерации" |
| Share sheet cancelled | Вернуть на Result Screen, ничего не менять |

### Edge Cases

| Ситуация | Поведение |
|----------|----------|
| Free user: PDF-экспорт заблокирован | "PDF-отчёты доступны с Premium" → [Подписаться] / [Поделиться изображением] (free) |
| Результат с RED danger level | Share image добавляет "SEEK IMMEDIATE HELP" + emergency number. PDF содержит жирный блок "Call 911 if bitten" |
| Старый скан из истории | "Поделиться" работает так же. Добавляется "Скан от [дата]" |
| Получатель шеринга открывает ссылку без приложения | Deep link → App Store page → скачивание. Attribution tracking для вирального роста |
