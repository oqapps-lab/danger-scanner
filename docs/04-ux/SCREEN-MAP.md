### 2. Main Tabs

#### 2.1 Home Tab

| Поле | Описание |
|------|----------|
| **Назначение** | Главный хаб: быстрый доступ к скану + обзор активности |
| **Откуда** | → 1.6 Paywall (первый раз) или → App launch (повторный) |
| **Куда** | → 2.2 Scan (кнопка "Сканировать"), → 2.2.5 Result (tap скана), → 5.1 Alert (tap алерта) |
| **Элементы** | |

**Состояние: Empty (нет сканов)**
- Приветствие "Привет, [имя]!" (или "Привет!" для guest)
- Статус: "Бесплатных сканов: 3/3" (Free) или "Premium" badge
- Empty state: иллюстрация + "Сделайте первый скан!" + список что можно сканировать (иконки 4 категорий с примерами)
- Кнопка [Сканировать] (крупная, по центру)

**Состояние: With Data (есть сканы)**
- Приветствие + статус подписки
- Секция "Сезонные алерты" (если есть для региона)
- Секция "Последние сканы" (горизонтальный scroll, до 5 карточек: thumbnail + danger level + дата)
- Кнопка [Сканировать] (floating, нижний правый или center tab)
- Статус сканов: "Осталось 2/3" (Free)

**Состояние: Premium**
- Без счётчика сканов
- Badge "Premium" в header
- Полные алерты (гео-сезонные)

#### 2.2.4 Post-Scan Questions

| Поле | Описание |
|------|----------|
| **Назначение** | Уточнение для более точной оценки опасности |
| **Откуда** | → 2.2.3 Analyzing |
| **Куда** | → 2.2.5 Result Screen (ответил или пропустил) |
| **Элементы** | 2–3 вопроса, каждый = экран с вариантами quick-tap. Адаптивны к категории (см. USER-FLOWS.md). Прогресс (1/3, 2/3, 3/3). [Пропустить] на каждом шаге |
| **Фича** | F-002 |

**Примеры вопросов по категориям:**

| Категория | Q1 | Q2 | Q3 |
|-----------|----|----|-----|
| Укус | "Где на теле?" (голова / руки / ноги / торс) | "Как давно?" (<1ч / 1–24ч / >24ч) | "Симптомы?" (отёк / покраснение / зуд / боль) |
| Паук | "Примерный размер?" (маленький / средний / большой) | "Где нашли?" (дом / гараж / двор / лес) | — |
| Змея | "Регион?" (auto из ZIP) | "Где встретили?" (двор / тропа / у воды) | "Был укус?" (да / нет / не уверен) |
| Клещ | "Где на теле?" (голова / шея / руки / ноги) | "Сколько мог быть прикреплён?" (<24ч / 24–72ч / >72ч / не знаю) | "Есть ли bullseye-покраснение?" (да / нет / не уверен) |

#### 2.2.5 Result Screen

| Поле | Описание |
|------|----------|
| **Назначение** | Показать оценку опасности + план действий |
| **Откуда** | → 2.2.4 Post-Scan Questions |
| **Куда** | → 4.6 Share (поделиться), → 2.1 Home (закрыть), → 4.5 Registration (если guest, после первого скана) |
| **Элементы** | |

**Верхняя секция:**
- Фото (thumbnail)
- Danger Level badge: RED ("Опасно") / YELLOW ("Будьте осторожны") / GREEN ("Низкий риск")
- "Похоже на: [название вида]" + confidence %

**Action Card:**
- 1. Немедленные действия (first aid инструкции)
- 2. Когда обращаться за помощью (ER / urgent care / врач / не нужно)
- 3. Что НЕ делать (список запрещённых действий)

**Disclaimer:**
- "Informational and educational only. Not a substitute for professional medical advice." (на каждом результате, F-010)
- Для змей: "If bitten by any snake, call 911 immediately" (всегда, F-004)

**Действия:**
- [Связаться со специалистом] (CTA, premium stub для MVP)
- [Сохранить результат]
- [Поделиться]

## Навигационная структура

```
Tab Bar (4 tabs)
├── Home      [icon: house]     → 2.1
├── Scan      [icon: camera]    → 2.2.1 (центр, увеличенная)
├── History   [icon: clock]     → 2.3
└── Profile   [icon: person]    → 2.4.1
```

**Stack Navigation внутри табов:**
- Home: Home → Seasonal Alert / Scan Detail
- Scan: Camera → Preview → Analyzing → Questions → Result
- History: List → Scan Detail
- Profile: Overview → Settings / Edit Profile / Subscription / Legal

**Modal Navigation:**
- Paywall (все варианты) — modal presentation (full screen)
- Share — action sheet (bottom)
- Permissions — modal (centered)
- Errors — alert (centered)
- Toast — overlay (top or bottom, auto-dismiss)
