## 1. Прямые конкуренты (идентификация опасностей)

### 1.1 Picture Insect: Bug Identifier

| Поле | Данные |
|------|--------|
| **Платформа** | iOS + Android |
| **Цена** | Freemium: $29.99/год после 7-дневного trial |

**Вывод**: Лидер среди insect ID-приложений, но не конкурирует на core use case Danger Scanner — danger assessment и action layer. Идентифицирует bug, но не оценивает опасность.

## 5. Gap Analysis — Что никто не делает

### 5.1 Критические gaps

| Gap | Текущее состояние рынка | Возможность для Danger Scanner |
|-----|------------------------|-------------------------------|
| **Multi-category danger ID** | Каждое приложение = 1 категория. Ни одно не объединяет даже 3 | Первый unified danger scanner. 8 категорий под одним брендом |
| **Danger assessment (red/yellow/green)** | Все конкуренты: только identification. «Это X вид». Никто не говорит «и вот что делать» | Каждый скан = danger level + action card + next steps |
| **One-tap specialist referral** | НОЛЬ конкурентов связывают пользователя с pest control / mold remediation / structural engineer | Monetizable action layer: $50–200/реферал |
| **Wall crack assessment** | **НОЛЬ consumer apps на рынке** | Абсолютный white space. $11.8B рынок ремонта фундаментов |
| **Community danger geo-map** | iNaturalist = biodiversity (не danger). TickTracker = только клещи | Unified danger map: «47 brown recluse отчётов в вашем ZIP за месяц» |
| **Instant tick Lyme risk assessment** | TickCheck = 2–5 рабочих дней (экспертная оценка). Нет instant AI | Мгновенная оценка вида клеща + вероятность заражения по региону |
| **Mold growth tracking** | Ни одно приложение не отслеживает изменения плесени со временем | Серия фото → «плесень выросла на 30% за месяц» |
| **Scan history + longitudinal tracking** | ChatGPT — неструктурированные чаты. Конкуренты — без истории или минимальная | Структурированная история: укусы, плесень, трещины — с timeline |

### 5.2 Что конкуренты могли бы скопировать (и за какое время)

| Feature | Кто может скопировать | Сложность | Время |
|---------|----------------------|-----------|-------|
| Добавить danger scoring к insect ID | Picture Insect | Низкая | 3–6 мес |
| Добавить plant toxicity | PictureThis | Низкая | 3–6 мес |
| Объединить 8 категорий | Любой новый стартап | Высокая | 12–18 мес |
| Referral network | Любой | Очень высокая | 12–24 мес (relationships) |
| Community geo-map с данными | Новый стартап | Средняя (tech), Высокая (data cold start) | 18–24 мес (до meaningful data) |

**Вывод**: Отдельные фичи легко скопировать, но **комбинация** (8 категорий + danger assessment + referral network + geo-map) создаёт моат, который требует 18–24 месяцев для репликации. First-mover advantage = реальный.

## 6. Market Saturation Score

### Оценка: **2/10** (рынок практически свободен)

**Обоснование:**

| Фактор | Оценка | Комментарий |
|--------|--------|-------------|
| Количество конкурентов в unified danger ID | 0/10 | **Ноль** прямых конкурентов. Категория не существует |
| Качество конкурентов в отдельных категориях | 2/10 | Spider ID = 2.6–3.1 stars. Mold apps = новорождённые. Wall cracks = пусто |
| Доминирующий игрок | 1/10 | PictureThis доминирует в plants, но не в danger assessment |
| Барьеры входа | 3/10 | Технические (8 моделей), но нет regulatory или patent barriers |
| Удовлетворённость пользователей | 2/10 | Reddit полон жалоб на существующие решения. Люди ждут часами на r/whatsthisbug |

### Почему 2, а не 1?

- PictureThis *мог бы* добавить toxicity assessment (растения)
- Picture Insect *мог бы* добавить danger scoring (насекомые)
- Google Lens продолжает развиваться
- Но: никто пока не двигается в эту сторону

### Что нужно для смены оценки?

- **3/10**: PictureThis добавляет «Is this plant toxic?» feature
- **4/10**: Крупный стартап объявляет multi-category danger app
- **5/10**: Google/Apple добавляет danger identification в нативные health features
- **7–8/10**: Два+ funded стартапа запускают конкурирующие unified danger apps

## 7. Ключевые выводы

1. **Danger Scanner создаёт новую категорию**, а не конкурирует в существующей. Это фундаментальное преимущество.

2. **Слабость конкурентов = возможность**: Spider ID apps (2.6 stars), mold apps (nascent), wall cracks (zero apps) — рынки ждут качественного решения.

3. **Единственная реальная угроза** — не текущие конкуренты, а:
   - PictureThis/Picture Insect решат расширить scope
   - Google/Apple интегрируют danger ID в нативные функции
   - Funded стартап с командой domain experts

4. **Моат строится через комбинацию**: 8 категорий + referral network + community geo-map + scan history. Каждый элемент по отдельности воспроизводим; комбинация — нет (18–24 месяца).

5. **Тайминг критичен**: Mold AI apps появились в 2024–2025. Рынок начинает оживать. Окно для first-mover = 6–12 месяцев.
